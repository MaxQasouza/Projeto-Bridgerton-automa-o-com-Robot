Projeto de **testes de API** realizado com base na **documentação Swagger** da aplicação – *bridgerton Academy*, como parte do programa de acadêmico da **QA.Coders Academy**.

>  Objetivo: praticar testes de API, integração e colaboração em equipe.

---

## Tecnologias utilizadas

-  **Linguagem:** JavaScript  
-  **Ferramenta:** Postman, Swagger

---

##  Pré-requisitos
Antes de começar, você precisa ter instalado:

- [Postman](https://www.postman.com/downloads/)

---

## Como Usar

1. Clone este repositório: https://github.com/Qa-Coders/funcional_squad_bridgerton.git

2. Abra o Postman.

3. Importe a coleção do projeto.

4. Execute as requisições conforme o fluxo definido.

## Estrutura do projeto 

```bash
funcional-squad-deploy-na-sexta/
|-- 00_auth.json
|-- 01_register.json 
|-- 02_user.json
|-- 03_admin.json
|-- 04_recovery.json
|-- nutrilife_env.json
|-- README.md
```
